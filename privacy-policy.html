Privacy Policy for Place ID finder

Last updated: April 17, 2026

Place ID finder reads the content of supported Search, Maps, and Travel pages automatically when those supported pages load in the browser.

What the extension does

The extension scans the current supported page to extract visible place ID values, displays the result to the user on that page, and lets the user copy the extracted value.

Data the extension accesses

The extension may access:

  • The current page URL on supported pages
  • The HTML content of the current supported page
  • Place ID values found on that page

How data is used

The extension uses page content only to extract and display place IDs for the current page.

Data storage

The extension does not store extracted place IDs, browsing history, or page content after the extraction is complete.

Data sharing

The extension does not send, sell, or share page content, place IDs, or browsing data with the developer or any third parties.

Remote code and external services

The extension does not execute remote code and does not rely on external servers to perform place ID extraction.

User controls

The extension runs automatically only on the supported page URLs declared in the extension manifest. It does not run on unrelated websites.
